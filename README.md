# The Cognitive Linguistics of Utopia
### Multimodal Metaphors and Conceptual Blending in Travis Scott’s Sonic World
‌
This repository contains the analytical framework, Multimodal Mapping Matrix (MMM), and lyrical corpus for the study of Travis Scott's *Utopia* (2023) through the lens of Cognitive Linguistics.
‌
## 🧠 Research Overview
This study integrates:
- **Conceptual Metaphor Theory (CMT)**
- **Conceptual Blending Theory (CBT)**
- **Force Dynamics (Talmy, 1988)**
‌
We explore how image schemas like `FORCE`, `PATH`, `CONTAINER`, and `VERTICALITY` are activated through sonic textures and lyrical narratives.
‌
## 📊 Key Concept: Utopian Vertigo
A core finding of this research is the identification of **"Utopian Vertigo"**—the cognitive state of experiencing aspiration (UP schema) alongside a constant awareness of potential collapse (DOWN schema/Deep Sub-bass).
‌
## 📁 Repository Structure
- `/data`: Contains the annotated lyrical corpus for tracks like "HYAENA", "SIRENS", and "MY EYES".
- `/matrix`: The Multimodal Mapping Matrix (MMM) correlating Hz/BPM with Image Schemas.
- `/scripts`: Python tools for identifying sonic-cognitive correlations.
‌
